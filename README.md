# Synthetic Image Compositing Demo

This repository demonstrates my workflow for creating realistic synthetic composites for AI training datasets and design projects.  

## Workflow
1. Collect raw photos (subway, street, product shots).  
2. Composite multiple layers using Photoshop/Illustrator.  
3. Apply AI-assisted scripts for refinement.  
4. Export clean, realistic final images.  

## Tools Used
- Adobe Photoshop / Illustrator  
- Python scripts for layer merging and dataset prep  
- AI-assisted generation tools
